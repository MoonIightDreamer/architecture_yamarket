# 1. Тема и целевая аудитория

## 1.1. Тема проекта
Сервис агрегации товаров и предложений интернет-магазинов с возможностью поиска, сравнения цен и оформления заказа (MVP Яндекс Маркета).

## 1.2. Обоснование ниши
Существуют крупные аналоги:
- Wildberries
- Ozon
- Avito

Это подтверждает наличие устойчивого спроса и большой аудитории.

## 1.3. Целевая аудитория
Пользователи, совершающие покупки в интернете.

Размер аудитории:
- ~100 млн пользователей в месяц
- ~10 млн пользователей в день

География:
- Европа и СНГ

## 1.4. MVP функционал
1. Поиск товаров
2. Просмотр карточки товара
3. Сравнение предложений продавцов
4. Добавление товара в корзину
5. Оформление заказа
6. Просмотр истории заказов

## 1.5. Ключевые продуктовые решения
- Централизованное хранение каталога товаров
- Агрегация предложений от разных продавцов
- Оформление заказа внутри платформы
- Асинхронное обновление цен
- Упрощенная персонализация выдачи
